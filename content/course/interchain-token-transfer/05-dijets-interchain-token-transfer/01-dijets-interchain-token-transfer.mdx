The Dijets Interchain Token Transfer is an application that allows users to transfer tokens between Dijets ECCs. The bridge is a set of smart contracts that are deployed across multiple Dijets ECCs, and leverages [InterECC Messaging](https://github.com/Dijets-Inc/teleporter) for cross-chain communication.

# What you will learn

In this chapter you will learn:

- **Bridge Design:** How the bridge is designed on a high level.
- **File Structure:** The structure of the bridge contracts and their dependencies.
- **Token Home:** What the token home is and how it works.
- **Token Remote:** What the token remote is and how it works.
